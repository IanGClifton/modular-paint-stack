# The Modular Paint Stack

The Modular Paint Stack (MPS) is a modular, parametric paint storage system for 3D printing. This project consists of multiple SCAD files to generate printable models for a substantial range of paint tubes, bottles, and brushes. Plus, shelves are available that support the Gridfinity system, allowing for even more flexibility. 

Please see the wiki for more details.
